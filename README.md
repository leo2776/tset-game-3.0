# TouchTestGame

這是一款觸控 / 滑鼠測試遊戲，用來測試電腦或觸控螢幕的反應能力。  
遊戲會生成不同圓圈，玩家需要按照提示「短按」或「長按」，一關比一關難。

## 功能特色
- 短按 / 長按 圓圈測試
- 一關一關增加難度
- 鼠標或觸控滑動軌跡
- 可播放背景音樂
- 打包成 Windows 可執行檔（.exe），不需安裝 Python

## 遊戲操作
1. 點擊圓圈表示「短按」，長按圓圈表示「長按」。
2. 完成所有圓圈後，自動進入下一關。
3. 每關圓圈數量和長短按要求隨機生成。

## 安裝與執行
1. 下載 Release 中的 `TouchTestGame.exe`。
2. 直接雙擊執行即可。

## 開發資訊
- 語言：Python 3
- GUI：Tkinter
- 音樂播放：pygame
- 背景圖片/音樂可自行替換
